## 主函数内部建议使用清晰的中文代码块注释：

```python
# ==================== 1. 实验参数 ====================

# ==================== 2. 被试与条件循环 ====================

# ==================== 3. 交叉验证 ====================

# ==================== 4. 训练集建模 ====================

# ==================== 5. 测试集测试 ====================

# ==================== 6. 单个被试条件结果汇总 ====================

# ==================== 7. 结果输出 ====================
```

## 写代码时的注意事项

1. 变量名和函数名要使用有含义的英文单词。可以使用常见缩写，但必须在注释中写清楚变量含义。

2. 不要使用没有意义的数字编号后缀。例如不建议使用：

```python
data1
data2
temp3
result4
```

更推荐：

```python
train_data
test_data
fold_accuracy
subject_result
```

3. 避免函数层层嵌套。只有当一段代码会被多个地方复用，或者逻辑本身比较独立时，才包装成函数。

4. 每个函数都要写清楚中文注释或文档字符串，说明输入、输出和作用。

5. 所有注释都使用中文，避免英语不好的人看不懂。

6. 比较复杂的代码必须补充解释性注释，尤其是：

```text
不常用的库
不常用的函数
不常见的写法
```
